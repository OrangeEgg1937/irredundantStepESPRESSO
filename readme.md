# Teseted Environment

# Compiler
- g++ (Ubuntu 11.4.0-1ubuntu1~22.04) 11.4.0


# How to run
1. Complie the file with g++

> g++ reduce.cpp -o reduce

2. Run the program with the following arguments

> ./reduce [input_path] [output_path]

For example, ./reduce gen0_in.txt output.txt
